# LinkNamuExtension
링크나무 확장프로그램입니다.

<br>

## 적용법
1. [chrome://extensions/](chrome://extensions/) 에서 개발자 모드로 변경
2. '압축해제된 확장프로그램을 로드합니다.' 클릭
3. 레포지토리에서 다운로드한 파일을 선택하여 확장프로그램에 추가

<br>
